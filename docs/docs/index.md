# E-Commerce Product Detail Page

A **MercadoLibre-style** e-commerce product detail page built with a modern full-stack architecture.

## Overview

This project implements a responsive product detail page that mimics the look and feel of Mercado Libre's product pages. It features a complete backend API serving product data, and a React frontend rendering a polished, responsive UI.

## Key Features

- **Product image gallery** with thumbnail navigation
- **Dynamic pricing** with discount badges and installment plans
- **Payment methods** display with card icons and installment info
- **Seller information** with official store badges
- **Customer reviews** with star distribution charts
- **Stock & shipping** indicators
- **Responsive design** — mobile, tablet, and desktop
- **RESTful API** with OpenAPI documentation

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | React 18 + TypeScript + Vite |
| Backend | FastAPI (Python 3.10+) |
| Database | SQLite via SQLAlchemy |
| Testing | pytest + httpx (backend) |
| Docs | MkDocs Material |
